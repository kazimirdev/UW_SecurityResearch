# 01 — Cel i przygotowanie

- Celem było wystawienie publicznego VPS na IPv4 jako pełnoportowego honeypota TCP/UDP.
- Środowisko: Debian 13, 1 GB RAM, 1 vCPU, pełna ekspozycja internetowa.
- Logowanie realizowano przez `nftables`, `systemd`, backend honeypot oraz realtime CSV.
- Pliki konfiguracji a dane znajdują się w `vps/`.
- Eksperyment trwał się 170 godziń.

- Instalujemy następne deb pakiety:
```sh
sudo apt update && sudo apt install -y \
nftables \
netcat-openbsd \
rsyslog \
systemd \
cron \
curl \
wget \
geoip-bin \
geoip-database \
dnsutils \
gawk \
coreutils \
sed \
grep \
awk \
procps \
iproute2 \
jq
```
- i skrypty z następnej części.
